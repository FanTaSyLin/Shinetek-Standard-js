## 基本规则

### JS

* 区分大小写
* JS中变量驼峰命名(常量的形式如: NAMES THIS)
* 使用单引号
* 不混用空格和Tab
* 去掉行末的空格(vs code快捷键ctrl k ctrl x)
* 同一行上插入大括号

### HTML

* 一般小写
* 命名一般使用 abcd-efgh
* 使用双引号

### CSS

* 一般小写
* 命名一般使用 abcd-efgh
* 使用双引号
